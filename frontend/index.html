The Project

Built to solve a real problem

🚧
Problem

Static Signals Fail Cities

Traditional traffic lights run on fixed timers — completely blind to actual traffic density. This wastes green time on empty roads and holds up packed lanes unnecessarily.

  • Fixed cycles ignore real-time congestion
  • Emergency vehicles get stuck in regular queues
  • No data feedback loop for optimisation
🧠
Solution

Vision-Based Adaptive Control

Live camera feeds from each lane are processed by YOLOv8 to count vehicles every 100 ms. Signal timings adjust proportionally, and emergency vehicles get instant green corridors.

  • YOLOv8 detects and counts vehicles per lane
  • Firebase syncs signal state to the dashboard
  • Emergency override clears all lanes instantly
🎯
Goal

Smarter, Safer Intersections

A deployable, low-cost system any city can adopt using IP cameras and a single server — reducing urban congestion and emergency response times with no infrastructure overhaul.

  • Reduce average wait time at intersections
  • Shorten emergency vehicle response time
  • Scale to multi-intersection city networks
